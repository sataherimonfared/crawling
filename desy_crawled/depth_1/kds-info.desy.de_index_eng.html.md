## KDS News

Since the beginning of October runs on the KDS Release 9.8. On the application side, there are no significant changes due to the new release. This release changes was a requirement for the installation of the new Web Gis. With Web Gis route infrastructure can be documented geographically on the DESY environment. Web Gis was also installed in early October and can now be used. Please send any kind of problems with the upgrade with a short description to kds-support@desy.de .
