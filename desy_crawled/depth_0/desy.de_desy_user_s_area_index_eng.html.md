# DESY USER

Information for DESY staff, scientific user groups and guests at the DESY sites in Hamburg and Zeuthen

## What does DESY USER stand for?

Offers and information on the DESY USER page are aimed at employees and users of DESY’s research infrastructure. Access to internal information is password-protected.
